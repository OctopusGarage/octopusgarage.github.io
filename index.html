A garage for durable AI work.

OctopusGarage builds compact open-source tools for the layer around coding agents: remote control, machine upkeep, source capture, private memory, and focused browser-native products.

What this is

How projects are judged

Local first. Operator visible. Useful before impressive.

Private data and working state should stay on the machine whenever possible.

Automation should expose intent, output, and approval points.

Each repository should solve one concrete workflow with a small public surface.
