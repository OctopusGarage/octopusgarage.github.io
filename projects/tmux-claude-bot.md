# tmux-claude-bot

> Control Claude Code and OpenAI Codex agents from Telegram, Feishu/Lark, a CLI, or a TUI.

## Links

- GitHub: https://github.com/OctopusGarage/tmux-claude-bot
- Releases: https://github.com/OctopusGarage/tmux-claude-bot/releases/latest

## Summary

tmux-claude-bot keeps terminal-based coding agents reachable after they start running. It connects tmux sessions to chat and terminal interfaces so an operator can send prompts, inspect output, manage multiple projects, and keep long-running work visible.

## Best For

- Remote control of Claude Code and OpenAI Codex sessions.
- Multi-project agent workflows that need persistent tmux panes.
- Developers who want chat access without losing the raw terminal session.
