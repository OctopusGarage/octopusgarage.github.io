# mbti-lab

> Bilingual MBTI personality test with offline support and shareable results.

## Links

- GitHub: https://github.com/OctopusGarage/mbti-lab
- Demo: https://octopusgarage.github.io/mbti-lab/

## Summary

mbti-lab is a browser-native personality test with Chinese and English support, sixteen-type result pages, shareable outcomes, and offline-friendly behavior. It is a compact example of a polished client-side assessment product.

## Best For

- Bilingual assessment product experiments.
- Shareable personality result pages.
- Vue-based browser products hosted on GitHub Pages.
