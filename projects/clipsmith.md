# clipsmith

> Capture posts, articles, media, and OCR into portable local bundles.

## Links

- GitHub: https://github.com/OctopusGarage/clipsmith
- Product: https://octopusgarage.github.io/clipsmith

## Summary

clipsmith is a CLI and agent-skill toolkit for saving web articles, social posts, media, and OCR output into portable local bundles. It is built for research, source collection, and repeatable capture workflows used by AI coding agents and local knowledge tools.

## Best For

- Capturing web and social media source material.
- OCR-backed local research bundles.
- Reusable Claude Code and Codex capture skills.
