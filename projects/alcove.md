# alcove

> Local-first personal memory for agent workflows.

## Links

- GitHub: https://github.com/OctopusGarage/alcove
- Product: https://octopusgarage.github.io/alcove/

## Summary

alcove is a local-first personal memory system for agent workflows. It gives builders a private knowledge base that can support retrieval and context reuse without turning working memory into a cloud-first product dependency.

## Best For

- Private personal memory around agent-assisted work.
- Local-first knowledge capture and retrieval.
- MCP-connected workflows that need durable local context.

## Keywords

Personal memory, knowledge base, local-first, MCP, PKM, Python, agent workflows.
