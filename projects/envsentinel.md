# envsentinel

> Audit local environment signals for AI coding CLIs.

## Links

- GitHub: https://github.com/OctopusGarage/envsentinel

## Summary

envsentinel is a terminal-first environment audit tool for AI coding CLIs. It checks local signals such as shell, DNS, network, security, and macOS setup so builders can spot machine problems before they affect long-running coding sessions.

## Best For

- Local environment audits for AI coding tools.
- Terminal-first developer diagnostics.
- Checking machine health before long agent runs.

## Keywords

Environment audit, AI coding, CLI, security, DNS, macOS, Go, developer tools.
