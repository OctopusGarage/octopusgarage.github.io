# git-auto-sync

> Auto-sync local repositories with AI-assisted commits, scheduling, and notifications.

## Links

- GitHub: https://github.com/OctopusGarage/git-auto-sync

## Summary

git-auto-sync monitors local repositories, stages changes safely, writes conventional commit messages with AI assistance, pushes to remotes, and sends notifications. It is designed for people who manage many local workspaces and want steady Git hygiene without hiding what changed.

## Best For

- Personal multi-repository sync workflows.
- AI-assisted conventional commit messages.
- Scheduled background pushes with visible logs and notifications.

## Keywords

Git automation, local repositories, AI commits, scheduler, Python, Telegram, Lark.
